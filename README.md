# php-flower-shop

## Run Locally (XAMPP)
1. Extract `flowershop-complete.zip` to `C:\xampp\htdocs\flowershop\`
2. **XAMPP:** Start Apache + MySQL (port 3307)
3. phpMyAdmin → Import `flowershop_db.sql`
4. Browser: **http://localhost/flowershop/index.php** ✅
